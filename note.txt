Tiap fungsi untuk berkomunikasi server dan client harus:
    Client:
    - dimulai send() dan diakhiri dengan read()

    Server:
    - dimulai dengan read() dan diakhiri dengan send()

Program:
    - server mengirim 0 jika query tidak valid
    - jika query valid maka akan mengembalikan message
    - message bisa message success ataupun messagae gagal

Kesalahan yang sering dilakukan:
- Tidak memanggil fclose() setelah fopen()
